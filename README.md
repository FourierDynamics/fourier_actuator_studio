# fourier_actuator_studio

## Project run

### Ubuntu
在终端执行下面命令，安装成功后在软件列表中双击运行

```
sudo dpkg -i fas_amd64.deb
```
---
### Windows
fas-x64-setup.exe 双击运行

---

### MacOS
fas.dmg 双击安装，成功后在应用程序中运行
